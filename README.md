# Python-TypeScript Scaffolding Framework

This is a framework for scaffolding Python-TypeScript projects. It is a minimalistic to-do app.

## Features

- Backend
  - FastAPI
  - SQLAlchemy
  - SQLite
  - Pytest
  - black
  - isort
  - mypy
- Frontend
  - React
  - TypeScript
  - Material-UI
- Makefile
